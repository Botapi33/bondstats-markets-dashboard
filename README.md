# Bond Market Dashboard

A live GitHub Pages dashboard for tracking sovereign 10-year government bond markets across countries.

## What this tool does

This dashboard provides a live global overview of sovereign bond markets using the BondStats global yield feed.

It shows:

- highest current 10Y yields
- largest daily yield increases
- largest daily yield declines
- average global yield
- benchmark sovereign markets
- top 10 highest-yield countries
- regional yield averages
- live spreads versus the US
- full market table

## Live data source

The dashboard reads from:

`https://botapi33.github.io/bondstats-global-yields/global_yields.json`
